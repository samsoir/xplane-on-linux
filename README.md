# xplane-on-linux

A personal collection of tools, configuration, and utilities for running and managing [X-Plane](https://www.x-plane.com/) on Linux.

This repository is a dumping ground for bits and pieces I've created or gathered over time — scripts, config snippets, tweaks, and notes — kept here so I don't lose them, and shared publicly in case any of it is useful to someone else.

## Scope

Contents may include (as they accumulate):

- Install and launch helpers
- System and hardware configuration notes (controllers, GPUs, audio, etc.)
- Performance tuning tips
- Plugin and add-on management utilities
- Troubleshooting notes

## Status

Informal and ad-hoc. Things land here when I find them useful; nothing is guaranteed to be polished, complete, or maintained. Use at your own risk.
